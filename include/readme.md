# SDL2 files setup

## Setup

Place SDL2 include headers in this directory: `/include/SDL2/`

## Required Headers

The following SDL2 headers are required:

- SDL.h
- SDL_image.h
- SDL_ttf.h
- SDL_mixer.h

## Directory Structure

```
include/
└── SDL2/
    ├── SDL.h
    ├── SDL_image.h
    ├── SDL_ttf.h
    └── SDL_mixer.h
```
